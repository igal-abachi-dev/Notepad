# Assets

Place your notepad.ico file here for the application icon.

You can create one from an SVG or PNG using tools like:
- ImageMagick: `convert icon.png -define icon:auto-resize=256,128,64,48,32,16 notepad.ico`
- Online converters

Or download a suitable icon from sites like:
- https://www.flaticon.com
- https://icons8.com
